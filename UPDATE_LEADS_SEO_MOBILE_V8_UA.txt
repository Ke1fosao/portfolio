ОНОВЛЕННЯ V8 — ЗАЯВКИ, SEO, CSS, МОБІЛЬНА ВЕРСІЯ

1. ЗАЯВКИ
- Telegram та email-сповіщення через backend/.env.
- Заявка спочатку зберігається у базі, тому не втрачається при помилці API або SMTP.
- Статуси: Нова, Переглянута, У роботі, Завершена.
- Автоматичний статус «Переглянута» після відкриття у React-адмінці.
- Індикатори Telegram/email та повторне надсилання з адмінпанелі.
- Honeypot, мінімальний час заповнення, валідація контактів, блокування дублів і rate limit.

2. СЕКРЕТИ
- Живих токенів і паролів в архіві немає.
- Запусти configure_notifications.bat та заповни backend/.env.
- Токен, який надсилався у чат, потрібно перевипустити в @BotFather.

3. SEO
- title, description, canonical, Open Graph, Twitter Card.
- JSON-LD Person, Service, Article, CreativeWork, BreadcrumbList.
- Окреме OG-зображення BABY LAND.
- sitemap.xml та robots.txt генеруються під час npm run build.
- Для production зміни VITE_SITE_URL і PUBLIC_SITE_URL на реальний домен.

4. CSS І МОБІЛЬНА ВЕРСІЯ
- Стилі сторінок розділені на окремі lazy chunks.
- Основний CSS зменшено приблизно до 141 КБ до gzip.
- Нова нижня мобільна навігація та анімоване меню.
- Мобільний режим навігації працює до 820 px, включно з планшетною шириною 768 px.
- Кнопка «Вгору» не перекриває нижню панель.

5. УМОВИ РОБОТИ
- Окрема сторінка /work-terms.
- Початок, передоплата, правки, передача, підтримка, матеріали від клієнта та права.
- Щоб прибрати: видали маршрут у frontend/src/App.jsx, посилання у Layout.jsx і файл WorkTerms.jsx.
