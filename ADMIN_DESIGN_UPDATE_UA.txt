ОНОВЛЕННЯ ДИЗАЙНУ АДМІНПАНЕЛІ
Дата: 06.07.2026

Що оновлено:
- Повністю уніфікований візуальний стиль усіх сторінок адмінпанелі.
- Нові картки, заголовки, таблиці, форми, кнопки, стани наведення та адаптивність.
- Усі стандартні select замінено кастомним dropdown-компонентом.
- Усі системні confirm/prompt/alert замінено кастомними модальними вікнами.
- Додано глобальний центр toast-повідомлень для успіхів, помилок і попереджень.
- На сторінках редагування додано єдину округлену плаваючу панель «Зберегти / Скасувати».
- Додано відстеження незбережених змін і захист від випадкового перемикання сторінок/записів.
- Оновлено CRM заявок, SEO, медіабібліотеку, візуальний редактор, системні сторінки та розділи «Про мене».
- Production-збірку перевірено командою npm run build — помилок немає.

Запуск фронтенду після розпакування:
1. cd frontend
2. npm install
3. npm run dev

Production-збірка:
1. cd frontend
2. npm run build

Папки node_modules і backend/venv навмисно не включені в архів: вони створюються локально та залежать від операційної системи.
